# Safety Eyewear Savings Calculator

Interactive calculator that shows businesses the true cost of workplace eye injuries and how much they can save with a managed safety eyewear program.

## How it works

1. **Enter your workforce size** — just one number
2. **Pick your industry** — sets all benchmarks automatically
3. **See Part 1** — the real cost of having no program (injuries, OSHA penalties, lost productivity)
4. **See Part 2** — DIY program vs. managed program, with hidden cost breakdown and annual savings

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- All calculations use published industry averages (BLS injury rates, OSHA penalty data, typical program benchmarks)
- Results are directional planning estimates, not company-specific forecasts
