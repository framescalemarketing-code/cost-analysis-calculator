# Cost Analysis Calculator

Next.js app for the Safety Eyewear Program Cost Analysis experience.

## What it does

- Injury Cost Context calculator
- Program Cost Comparison calculator
- Models annual current-state vs managed-model total costs
- Estimates annual savings and payback window

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- Default values include national context figures previously discussed:
  - $5,800 medical + $1,690 work loss per nonfatal injury
  - $159,358 per occupational vision loss incident
- Results are directional planning estimates, not company-specific forecasts.
